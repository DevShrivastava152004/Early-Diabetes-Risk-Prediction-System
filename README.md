# 🩺 EARLY DIABETES RISK PREDICTION

An explainable Machine Learning based web application that estimates the risk of diabetes using health indicators and provides user-friendly health recommendations.


## 🚀 Live Demo

🔗 [Click Here to Use the App](https://early-diabetes-risk-prediction-system.streamlit.app/)


## 📌 Problem Statement

- Diabetes is a growing global health concern
- Early Detection is important
- Flase negatives in medical screening are dangerous
- There ia a need for an accessible early-risk estimation tool.

## 💡 Solution Overview

- Built using Logistic Regression for probabilistic prediction.
- Implemented features interaction (Glucose  X BMI).
- Applied threshold optimization to improve recall.
- Desgined risk segmentation for better interpretation.
- Integrated rule-based explanation engine.
- Deployed using Streamlit Cloud.

## 🧠 Model & Technical Details

- Dataset used
- Number of features
- Feature scaling using StamdardScaler
- AUC score achieved(0.81)
- Why Logistic Regression was selected
- Why threshold was reduced to 0.24
- Importance of Recall in medical screening

## 🖥️ Application Features

- Senior-friendly large font interface
- Risk categorization (Low, Mild, High, Critical)
- Plain-Language heatlh summary
- Actionable lifestyle recommendation
- Downloadable risk report
- Public deployment


## ⚙️ How to Run Locally

- Clone the repository
- Create a virtual enovironment
- Install dependencies using `pip install -r requirements.txt`
- Run `streamlit run app.py`
